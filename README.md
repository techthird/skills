# Skills
> Agent Skills 公共资源库-汉化版

Skills 是一种可移植、受版本控制的包，用于教会 Agent 如何执行特定领域的任务。

与Prompt的区别，Prompt可以理解为“一句话命令”或“即时指令”，作用范围仅限于当前单次请求/回合，特点是灵活多变，每次都可以不同。

- 可移植：技能可以在任何支持 Agent Skills 标准的 Agent 中反复使用。
- 版本控制：技能以文件形式存储，可以在你的代码仓库中进行管理和追踪，或通过 GitHub 仓库链接安装。
- 特点：可完成更复杂任务，封装特定逻辑、调用指定API、工具脚本。

# 目录结构
```
.claude/
└── skills/
    └── my-skill/
        └── SKILL.md
```

# 使用
```shell
/my-skill 用户输入内容
```
